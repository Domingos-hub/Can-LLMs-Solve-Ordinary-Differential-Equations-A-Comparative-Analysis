# Can-LLMs-Solve-ODEs?-A-Comparative-Analysis
This paper evaluates  DeepSeek-R1, Claude-Haiku- 4.5, Llama-3.3-70B-Instruct, Gemini-2.5-Pro, and GPT-5.2, using ten first- order ODE application problems and a task involving analogous problem generation. We propose a multi-dimensional framework assessing ODE modeling, Python code generation, accuracy, and the generation of analogous problems.
